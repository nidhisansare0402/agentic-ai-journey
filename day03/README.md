# Smart AI Assistant

## Project Description

A rule-based AI Assistant built using Python and the Gemini API.

The application decides whether to use local tools or Gemini based on the user's request.

## Features

- Calculator Tool
- Word Counter
- Text Statistics
- Current Date & Time
- Gemini Integration

## Technologies Used

- Python
- Google Gemini API
- python-dotenv

## Folder Structure

app.py          - Main application

chatbot.py      - Gemini interaction

tools.py        - Local tools

config.py       - Configuration

prompts.py      - Prompt templates

utils.py        - Helper functions
